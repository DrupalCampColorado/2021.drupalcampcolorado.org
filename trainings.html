On August 6, grow your skillset in one of three trainings

Join us the morning of Friday, August 6, for one of three trainings with experts from the Drupal community. Each training will go deep to help you build the skills you need as a web professional.

Date: Friday, August 6, 2021
Time: 9:00am - 12:00pm (MT)
Location: Virtual / Online

Component Development with Emulsify

Come learn component development. Learn ATOMIC Design but with Drupal in mind. While we are in there, we'll look at the TailwindCSS framework and how to use it in components and keep the code semantic.

Learn more

DDEV-Local: From Beginner to Expert

DDEV is a docker-based local development environment that works on macOS, Windows and Linux... and works the same everywhere. It's easy to get started, and it's not hard to get into advanced things as well.

Learn more

Upgrading to Drupal 9 Using the Migrate API

Drupal 9 is out! Learn to use the Migrate API to upgrade your Drupal 6/7 site to Drupal 8/9. In this training we will teach you what tools are available to accomplish the task and how to use them. You will also learn how to plan and execute successful upgrade projects.

Learn more
